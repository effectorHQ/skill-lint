CLI COMMANDS
================================================================================

1. openclaw init <name>
   - Scaffolds a new plugin with complete project structure
   - Creates: src/, tests/, pyproject.toml, README.md, LICENSE
   - Generates boilerplate plugin code with PluginBase pattern
   - Automatic snake_case conversion from plugin name

2. openclaw validate [path]
   - Validates plugin structure and configuration
   - Checks: required files, directory structure, TOML syntax
   - Tests: package imports, test structure
   - Returns: clear pass/fail with detailed reporting

3. openclaw test [path]
   - Runs pytest with OpenClaw-specific settings
   - Supports custom pytest arguments via --args flag
   - Wraps pytest for integrated testing experience

4. openclaw dev [path]
   - Starts local development mode with file watching
   - Supports auto-reload with watchfiles (with fallback)
   - Automatic module reloading on source changes
