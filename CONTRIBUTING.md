# Contributing

See the [effectorHQ Contributing Guide](https://github.com/effectorHQ/.github/blob/main/CONTRIBUTING.md) for the full contribution process, code standards, and PR checklist.

## Quick start

```bash
git clone https://github.com/effectorHQ/skill-lint.git
cd REPO_NAME
npm install
npm test
```

## Issues and pull requests

- Open an [issue](../../issues) before starting significant work
- All PRs require passing tests (`npm test`) and a clear description
- Follow the existing code style — zero dependencies, ESM-only, Node ≥ 18
